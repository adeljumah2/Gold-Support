This project, in its simplest form,

is a gold price analysis tool to help you buy or sell gold based on its historical price patterns.

The following tools were used in this project:

1- Web Scraping (Requests, PyQuery)
2- Pandas Data Manipulation
3- Scikit-Learn Preprocessing (MinMaxScaler)
4- JSON File Handling
5- Multithreading
6- Web-based Interface (HTML, CSS, JavaScript)

Note: This project does not predict gold prices; it only analyzes historical gold prices.
